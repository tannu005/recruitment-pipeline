# 🚀 AI-Powered Recruitment Pipeline

## Architecture
- **Backend**: Node.js + Express + SQLite + GraphQL (Apollo)
- **Frontend**: React + TypeScript + Vite + TailwindCSS
- **AI Engine**: Gemini Pro for resume evaluation
- **Real-time**: Socket.IO for live updates
- **Workspace**: Jetro canvas for visual analytics

## Key Features
- PDF resume parsing & AI-powered evaluation
- Multi-dimensional scoring (Skill, Experience, Culture, Risk)
- Real-time WebSocket notifications
- GraphQL API for flexible queries
- Role-based auth with JWT
- Audit logging

## Data Pipeline
`Resume Upload → Text Extraction → AI Scoring → Ranking → Dashboard`
